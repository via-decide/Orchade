# Audio Direction

Build adaptive music layers for calm care, storm pressure, market/village, caves, and success. Environmental ambience should react to weather, biome, time, and ecology health. Interaction feedback needs distinct water, harvest, plant, sell, craft, error, unlock, and save cues. NPC voices can use short phoneme/texture barks with captions. Dynamic mixing should duck ambience under important UI and expose volume sliders. Accessibility audio should provide optional readiness pings, danger alerts, and stereo-independent cues.
