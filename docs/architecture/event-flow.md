# Event Flow

## User Events

- Authentication: Google sign-in, guest sign-in, logout.
- Gameplay: plant, water, apply tools, advance day, harvest.
- Navigation: switch active tab or selected orchard.
- Social/economy: transfer credits or data seeds, read rankings.

## Platform Events

- Startup and initial render telemetry.
- Browser `error` and `unhandledrejection` telemetry.
- Browser `online` and `offline` telemetry.
- Firestore snapshot updates.
