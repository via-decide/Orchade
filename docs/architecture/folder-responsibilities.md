# Folder Responsibilities

| Folder | Responsibility | Owner |
| --- | --- | --- |
| `src/` | Runtime app shell, game state orchestration, Firebase integration | Gameplay / Engine |
| `src/components/` | Presentational and interactive UI components | UI |
| `src/lib/` | Cross-cutting engineering platform utilities | Engine |
| `docs/` | Human-readable engineering state and decisions | Docs |
| `docs/architecture/` | System maps and flow documentation | Architecture |
| `.github/` | GitHub-native automation, templates, ownership | Infrastructure |
| `scripts/` | Repeatable local and CI repository automation | Infrastructure |
| `game-repo/` | Duplicated project snapshot pending ownership decision | Blocked |
