# UI/UX Engineering

Audit priorities: HUD must separate immediate care status from long-term economy; inventory needs sorting/filtering/compare; menus need keyboard/gamepad parity; navigation needs breadcrumbs and clear back behavior; tooltips must explain effects and consequences; controller/touch require command abstraction; mobile needs larger targets and simplified density; desktop can support richer panels; accessibility requires contrast modes, reduced motion, scalable text, remapping, captions, audio cues, colorblind-safe plant/weather signals, and reversible/descriptive errors.
