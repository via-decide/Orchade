# Player Journey

## Current journey evidence
Players begin with day 1, 100 credits, one unlocked nine-plot orchard, two locked orchards, empty harvest archive, initial upgrades, random weather, and tabs for orchard, lab, market, rankings, archives, and profile. The playable loop asks players to buy seeds, select plants, perform care actions, advance days, harvest, spend Data Seeds on upgrades, use market features, and unlock archives through harvested types.

## Estimate scale
