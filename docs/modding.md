# Modding Platform

Modding target: content packs first, safe scripting later. Provide manifest, semantic versioning, dependency resolution, schema validation, load order, ID namespaces, conflict reports, and save dependency records. Plugin API should expose read-only registries, event hooks, recipe/quest/crop definitions, localization, and sandboxed automation scripts with CPU/time limits. Steam Workshop readiness requires preview images, tags, version constraints, moderation metadata, and deterministic validation before upload.
